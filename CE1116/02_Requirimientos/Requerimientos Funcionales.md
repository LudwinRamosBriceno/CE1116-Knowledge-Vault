## 📚 Idea/Concepto 

Es el comportamiento que debe tener el sistema bajo ciertas condiciones, es lo que se debe implementar para permitir que los usuarios puedan interactuar con el sistema y lograr sus objetivos, y que por ende se cumpla el propósito del proyecto. Son las entradas y las salidas del sistema.
